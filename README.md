# Selenium Web Automation - OrangeHRM

## Introduction
This repository contains a Selenium Web Automation Test Suite developed in Java for the OrangeHRM platform. The suite automates various human resource management tasks, enhancing the efficiency and reliability of web-based HR operations.

## Features
- **Comprehensive Test Coverage:** Automated test cases for major functionalities of OrangeHRM.
- **Modular Structure:** Well-organized code for easy maintenance and scalability.
- **Cross-Browser Testing:** Supports multiple browsers for testing.
- **Detailed Reporting:** Generates detailed test reports for analysis.

## Technologies Used
- **Selenium WebDriver**
- **Java**
- **TestNG**
- **Maven**

## Setup Instructions

### Prerequisites
- Java JDK 8 or above
- Maven
- Any IDE (e.g., IntelliJ IDEA, Eclipse)
- Browsers (e.g., Chrome, Firefox)

### Installation
1. **Clone the repository:**
   ```sh
   git clone https://github.com/SourabhKokitkar/Selenium-Web-Automation-Java.git
   ```
2. **Navigate to the project directory:**
   ```sh
   cd Selenium-Web-Automation-Java
   ```
3. **Install dependencies:**
   ```sh
   mvn clean install
   ```

## Project Structure
- **src/main/java:** Contains the main codebase.
- **src/test/java:** Contains test cases.
- **resources:** Contains configuration files.
